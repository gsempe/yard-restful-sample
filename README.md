## yard-restful-sample

Sample application for [yard-restful plugin](http://github.com/kraft001/yard-restful)

## Installation

  ```
  bundle install
  bundle exec yard doc
  x-www-browser ./public/api_doc/index.html
  OR
  x-www-browser ./public/api_doc/frames.html
  ```
