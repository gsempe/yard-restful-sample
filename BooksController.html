Books

Books and articles

create

Create new book

URL

POST /books

Required Parameters

  • author_id Integer

    Identifier of an author
  • title String

    Title of the book

Optional Parameters

  • year Integer

    Imprint year
  • genre String

    Genre of the story

Response Object

  • Book

    The created book

Examples

Let's try to create a book Request 
{
  "author_id": 1,
  "title": "My first book",
  "year": 1999
}
Response 
{
  "author": {
    "name": "Petr Petrov",
    "birthdate": "1968-03-25"
  },
  "title": "My first book",
  "year": 1999,
  "genre": ""
}
The book should be created correctly
